About

Unicart is a fullstack e-commerce application to buy watch related products which was build using MERN Stack-MongoDB, ExpressJS, React and NodeJS

Technologies used

  • ReactJs
  • Mongodb
  • Express
  • React Router
  • Node
  • Postman
  • React-bootstrap

Features

  • OAuth functionality
